#  Expense Tracker
A modern and responsive **Expense Tracker Web App** built using **HTML**, **CSS**, and **JavaScript**.

This application helps users manage their income and expenses efficiently with features like transaction tracking, category filtering, search functionality, balance calculation, and dark/light theme support.

---
##  Features
-  Add income and expense transactions
-  Real-time balance calculation
-  Track total income and expenses
-  Add transaction dates
-  Category-based transaction management
-  Search transactions
-  Filter transactions by category
-  Dark / Light Theme Toggle
-  Fully Responsive Design

---
##  Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS)
